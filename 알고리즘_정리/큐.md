# 🔰 Queue        
> 1:1 선형구조     
> 선입선출(FIFO)     
 
### ✔ Queue<Integer> queue = new LinkedList<>();     
> size() : 원소 개수 확인     
> offer() : 뒤쪽에 원소 삽입       
> poll() : 가장 오래된 원소 삭제       
> creatQueue : 공백상태의 큐 생성     
> isEmpty() : 공백확인 inEmpty()     
> isFull : 포화상태인지 확인     
> Qpeek : 앞쪽에서 삭제없이 반환
